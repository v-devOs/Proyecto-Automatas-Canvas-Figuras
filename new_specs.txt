Funcionalidades del lenguaje
Nuevos comandos:

move <id> (dx, dy) — desplazamiento relativo (actualmente solo hay posición absoluta con update)
copy <id> — duplicar una figura existente con un nuevo ID
group <id1> <id2> ... / ungroup <gid> — agrupar figuras para moverlas/rotarlas juntas
scale <id> (factor) — escalar de forma relativa en lugar de absoluta
Nuevos tipos de figura:

rectangle con parámetros de ancho y alto independientes
ellipse con radio horizontal y vertical
text — colocar texto en el canvas


Expresiones en parámetros:

Valores negativos: create circle("red", 2, [-5, 10])
Decimales: create circle("red", 1.5, [0, 0])
Variables: set x 10 + create circle("red", 2, [x, y])

Múltiples comandos por línea separados por ;

Análisis semántico
Verificación de colores — actualmente cualquier string pasa como color válido; se podría validar contra una lista de nombres CSS o formato hex estricto
Límites del canvas — advertir (warning) si la figura queda fuera del área visible
Solapamiento — detectar figuras completamente superpuestas y advertirlo
Visualización / Canvas
Selección interactiva — hacer clic en una figura en el canvas para seleccionarla y ver/editar sus propiedades
Arrastrar figuras — mover figuras con drag & drop directamente en el canvas
Zoom y pan — escalar y desplazar el canvas con rueda del ratón
Exportar — guardar el canvas como imagen PNG/SVG
Animaciones — animate <id> move (dx, dy) steps 10 para transiciones suaves
Consola e interfaz

Deshacer/rehacer — undo / redo para revertir el último comando
Panel de propiedades — al seleccionar una figura mostrar un panel lateral con sus atributos editables
